Graduate Research Portfolio

M.S. Mechanical Engineering // University of Illinois Chicago

01. Exoskeleton Teleoperation

Human-Robot Interaction

Master-Slave Control Architecture

Designed a teleoperation interface mapping human joint movements to a humanoid robot in real-time. The system utilizes an exoskeleton arm acting as the "master" controller to drive the "slave" humanoid actuators.

  • Key Tech: Kinematic mapping, real-time signal processing.
  • Outcome: Achieved synchronous motion replication with low latency, enabling precise remote manipulation.

02. Animatronic Vision System

Computer Vision

Gaze Tracking with OpenCV & Arduino

Developed an animatronic eye mechanism capable of tracking human faces autonomously. The system integrates computer vision software with hardware actuation.

  • Software: Python, OpenCV (Haar Cascades/DNN) for face coordinates.
  • Hardware: Arduino microcontroller handling servo kinematics.
  • Integration: Serial communication bridge converts pixel coordinates (X,Y) into servo angles (Theta, Phi) in real-time.

03. Minitaur Leg Mechanism

Legged Locomotion

Constrained Kinematics Analysis

Conducted analysis on a 2-DOF five-bar linkage mechanism (Minitaur leg). Focused on the implementation of constrained hopping behaviors and stability analysis.

  • Focus: Raibert hopping controllers and stance phase dynamics.
  • Analysis: Mapped motor torques to toe forces using Jacobian transformations to optimize jump height and landing stability.

04. Zumobot Autonomous Navigation

Control Systems

PID Velocity Control & Perception

Engineered a closed-loop control system for a differential drive robot. Integrated a RealSense depth camera to detect drivable ground planes and avoid obstacles.

  • Control Loop: Implemented a PID controller using built-in encoders to ensure PWM input linearly correlates to actual wheel velocity, compensating for battery voltage drop and friction.
  • Perception: RealSense depth cloud processing for floor segmentation.
